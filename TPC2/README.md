# TPC2: Mapa Virtual

## Autor:
- a94877
- Diogo Cardoso Ferreira

## Resumo
Neste trabalho foi desenvolvido um servidor que faz requisições HTTP para obter os dados sobre
cidades e das ligações entre as mesmas.
Primeiro é criado um servidor web utilizando 'http' que fica à escuta na porta 2002 e responde aos pedidos HTTP.
O servidor responde às solicitações com páginas HTML geradas dinamicamente, com base nos dados obtidos por meio de requisições HTTP.
As requisições HTTP são feitas para obter dados sobre cidades e ligações entre elas, que são então utilizados para renderizar as páginas HTML.
O código é estruturado para lidar com diferentes caminhos de URL, fornecendo diferentes funcionalidades com base na solicitação do cliente.
